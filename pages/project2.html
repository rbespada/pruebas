Proyecto 2: Aplicación de Tareas

Descripción detallada del proyecto 2. App web para gestión de tareas usando React y Node.js, con base de datos MongoDB. Permite crear, editar y eliminar tareas, con autenticación de usuarios.

Tecnologías utilizadas

  • React
  • Node.js
  • MongoDB
  • Express

Características

  • Autenticación JWT
  • CRUD de tareas
  • Interfaz responsive
  • API REST
