# Portfolio Web Personal

Este es mi portfolio personal desarrollado para la Práctica 1.

## Tecnologías utilizadas
- HTML5
- CSS3 (con Flexbox y Grid para responsive)
- JavaScript (menú responsive y popup de cookies)

## Estructura
- `index.html`: Página principal
- `css/styles.css`: Estilos
- `js/scripts.js`: Scripts
- `pages/privacy.html`: Política de privacidad
- `memoria.md`: Memoria de la práctica

## Enlaces
- Repositorio: https://github.com/rbespada/pruebas
- Sitio web: https://rbespada.github.io/pruebas (activar GitHub Pages en Settings > Pages)

## Instalación
Clona el repo y abre index.html en un navegador.
