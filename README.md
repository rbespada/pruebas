# Portfolio Web Personal

Este es mi portfolio personal desarrollado para la Práctica 1.

## Tecnologías utilizadas
- HTML5
- CSS3 (con Flexbox, Grid y animaciones)
- JavaScript (menú responsive, popup de cookies, animaciones al scroll)

## Estructura
- `index.html`: Página principal con secciones
- `pages/project1.html`, `project2.html`, `project3.html`: Páginas detalladas de proyectos
- `pages/privacy.html`: Política de privacidad
- `css/styles.css`: Estilos con animaciones
- `js/scripts.js`: Scripts dinámicos
- `memoria.md`: Memoria de la práctica

## Características dinámicas
- Animaciones al scroll
- Navegación responsive
- Páginas dedicadas para proyectos

## Enlaces
- Repositorio: https://github.com/rbespada/pruebas
- Sitio web: https://rbespada.github.io/pruebas (activar GitHub Pages)

Clona el repo y abre index.html en un navegador.
