# 📁 Portfolio Web Personal

Portfolio moderno y responsivo con servidor Express.js, diseño Tailwind CSS y características dinámicas.

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor
npm start

# Abrir en navegador
# http://localhost:3000
```

## 🌍 Despliegue en Producción

### **🔷 Opción 1: VERCEL (Recomendado)**
1. Ve a [vercel.com](https://vercel.com)
2. Conecta tu GitHub (`rbespada/pruebas`)
3. ¡Se despliega automáticamente!
4. URL: `https://[tu-proyecto].vercel.app`

### **🟠 Opción 2: NETLIFY**
1. Ve a [netlify.com](https://netlify.com)
2. Conecta tu GitHub (`rbespada/pruebas`)
3. Netlify lee `netlify.toml` automáticamente
4. URL: `https://[tu-proyecto].netlify.app`

## 📋 Tecnologías Utilizadas

**Frontend:**
- HTML5 (estructura semántica)
- CSS3 + Tailwind CSS (CDN)
- JavaScript vanilla (ES6+)
- Material Symbols Icons
- Unsplash API (imágenes)

**Backend:**
- Node.js
- Express.js
- Compression middleware

## 🏗️ Estructura

```
├── index.html                  # Página principal
```
